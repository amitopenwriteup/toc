#  CKA Exam Training Plan — Table of Contents
**Level:** Intermediate | **Duration:** 20 Hours | **Split:** 80% Lab · 20% Theory

---

##  Module 1 — Cluster Architecture, Installation & Configuration
 Exam Weight: 25%*

### Theory
- 1.1 Kubernetes Control Plane Components
- 1.2 Node Components & Container Runtime
- 1.3 RBAC — Roles, ClusterRoles & Bindings
- 1.4 kubeadm Bootstrap Process & Certificate Management

### Labs
- 1.1 Cluster Bootstrap with kubeadm
- 1.2 etcd Backup & Restore
- 1.3 RBAC Configuration & Testing
- 1.4 Cluster Version Upgrade

##  Module 2 — Workloads & Scheduling
Exam Weight: 15%*

### Theory
- 2.1 Workload Resources — Deployments, DaemonSets, StatefulSets, Jobs, CronJobs
- 2.2 Resource Requests & Limits
- 2.3 Node Affinity, Taints & Tolerations
- 2.4 ConfigMaps & Secrets

### Labs
- 2.1 Deployments & Rolling Updates
- 2.2 Scheduling — Taints, Tolerations & Node Affinity
- 2.3 Resource Management & Quotas
- 2.4 ConfigMaps & Secrets in Pods

##  Module 3 — Services & Networking
Exam Weight: 20%*

### Theory
- 3.1 Service Types — ClusterIP, NodePort, LoadBalancer, ExternalName
- 3.2 Ingress Controllers & Ingress Rules
- 3.3 DNS Resolution & CoreDNS
- 3.4 NetworkPolicy Fundamentals

### Labs
- 3.1 Service Types & Connectivity
- 3.2 Ingress Configuration & Routing
- 3.3 NetworkPolicy — Deny & Allow Rules
- 3.4 CoreDNS & Service Discovery

##  Module 4 — Storage
 Exam Weight: 10%*

### Theory
- 4.1 PersistentVolumes & PersistentVolumeClaims
- 4.2 StorageClasses & Dynamic Provisioning
- 4.3 Access Modes & Volume Types

### Labs
- 4.1 Static PV & PVC Creation
- 4.2 Dynamic Provisioning with StorageClass

---

##  Module 5 — Troubleshooting 
Exam Weight: 30% — Highest Priority*

### Theory
- 5.1 Troubleshooting Methodology — Cluster → Node → Pod
- 5.2 Reading Logs — kubectl, journalctl, Runtime Logs
- 5.3 Common Failure Patterns & Fixes

### Labs
- 5.1 Pod Failure Scenarios — CrashLoopBackOff, ImagePullBackOff, OOMKilled
- 5.2 Node Troubleshooting — NotReady, kubelet Failures
- 5.3 Control Plane Troubleshooting — API Server, Scheduler, etcd
- 5.4 Networking Troubleshooting — Services, DNS, CNI
- 5.5 Full Scenario Simulation — Timed Practice
